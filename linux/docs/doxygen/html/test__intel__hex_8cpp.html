◆ parseIntelHex()

std::vector< HexRecord > pupradar::parseIntelHex ( std::istream & input)

Parses a stream of Intel HEX text and returns all data records (type 00) in file order.

Throws std::runtime_error on:

  • missing leading ':'
  • non-hex character
  • byte-count mismatch
  • checksum failure
  • unknown record type other than 00/01/02/03/04/05
  • missing EOF record (type 01)
